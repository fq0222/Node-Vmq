# database

## 目录说明

该目录用于存放数据库相关文件。

当前阶段已建立：

- `migrations`：迁移脚本目录
- `seeds`：初始化脚本目录

当前 `TP-02` 已补充：

- PostgreSQL 建表脚本
- 索引脚本
- 初始化默认配置脚本
- `tmp_prices` 重建脚本

## 使用说明

后端服务启动时会按顺序执行：

1. 数据库连接检查
2. `migrations` 目录中的 SQL 迁移
3. `settings` 表空表检测
4. 默认配置初始化

`seeds` 目录中的 SQL 文件主要用于人工排查、独立执行或部署时参考。
