# Node-Vmq

## 项目说明

Node-Vmq 是对原 `Vmq` 项目的现代化重构版本。

当前阶段已完成：

- 需求文档整理
- 接口兼容文档整理
- 数据库设计文档整理
- 开发任务拆分
- `TP-01` 工程基础设施骨架搭建
- `TP-02` 数据库与初始化骨架搭建

## 技术栈

- 后端：Node.js + Express
- 前端：Vue 3 + Vite
- 数据库：PostgreSQL

## 目录结构

```text
.
├─ apps
│  ├─ api            # 后端服务
│  └─ web            # 前端应用
├─ packages
│  └─ shared         # 共享常量与工具
├─ database
│  ├─ migrations     # 数据库迁移脚本
│  └─ seeds          # 数据库初始化脚本
├─ docs              # 需求、接口、计划文档
└─ original-vmq      # 原项目参考代码
```

## 工作区命令

在依赖安装完成后，可使用以下命令：

```bash
npm run dev:api
npm run dev:web
```

如果需要分别进入子项目执行，也可以：

```bash
cd apps/api
npm run dev

cd apps/web
npm run dev
```
